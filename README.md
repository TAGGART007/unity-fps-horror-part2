# Unity FPS Horror Game - Part 2

**FPS Horror Game Part 2** is the continuation of a 3D first-person horror game built in **Unity 3D**. Players navigate a dark environment, fight zombies, collect treasures, and survive using a gun, keyboard, and mouse.  

This part adds **new weapons, zombie types, levels, and improved mechanics**.

---

## Gameplay Features

- **First-Person Shooter Mechanics**
  - Move using **WASD** keys
  - Look around using **Mouse**
  - Fire weapons with **Left Click**
  - Reload weapons using **R**

- **Enemies**
  - Zombies with AI movement towards player
  - New zombie types with varying speed and health

- **Treasures & Objectives**
  - Collect treasure chests placed throughout levels
  - Completing all objectives allows the player to win

- **Levels & Difficulty**
  - New areas with increasing difficulty
  - More zombies and obstacles in later stages

- **Health & Damage**
  - Player has a health bar
  - Zombies deal damage on contact

---

## Tech Stack

- **Engine:** Unity 3D (2019/2020/2021 or later)  
- **Language:** C#  
- **Assets:** 3D models for player, zombies, environment, weapons  
- **Input Controls:** Keyboard + Mouse
